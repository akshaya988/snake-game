# 🐍 Snake Game

A classic Snake Game built using **HTML, CSS, and JavaScript**. The objective is to eat the food, grow the snake, and achieve the highest score without colliding with the walls or the snake's own body.

## 🎮 Live Demo

👉 https://akshaya988.github.io/snake-game/

## 📂 GitHub Repository

👉 https://github.com/akshaya988/snake-game

## 🚀 Features

- Classic Snake Game
- Smooth gameplay
- Score tracking
- Responsive game board
- Keyboard controls
- Simple and clean user interface

## 🎮 How to Play

1. Open the game in your browser.
2. **Press any key to start the game.**
3. Control the snake using the **Arrow Keys**:
   - ⬆️ Up Arrow
   - ⬇️ Down Arrow
   - ⬅️ Left Arrow
   - ➡️ Right Arrow
4. Eat the food to increase your score and make the snake grow.
5. Avoid hitting the walls or the snake's own body.
6. Try to achieve the highest score!
